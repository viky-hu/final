# Final

基于 **Monorepo** 架构的 Web 前端项目，通过「组件化拆分」先独立开发各功能窗口，再在主平台中整合。

## 🛠 技术栈

| 类别 | 技术选型 |
|------|----------|
| **管理工具** | pnpm workspaces + Turborepo |
| **主应用 (Host)** | Next.js 14+ (App Router) |
| **开发库 (Lab)** | React + Vite + TypeScript |
| **样式方案** | Tailwind CSS |
| **基础组件** | shadcn/ui (Radix + Tailwind) |
| **动画与特效** | Magic UI + Motion (Framer Motion) |
| **核心动画引擎** | GSAP — 窗口弹出、拖拽、视差滚动 |
| **交互式动画** | Rive — 导航菜单、图标、品牌展示（参考 Dropbox Brand） |
| **组件规范** | Vercel 最佳实践 (Server Components 优先) |

### 为什么选择 GSAP？

GSAP 是 Awwwards 获奖作品的标配，大量顶级网站用它实现丝滑动画。在本项目中：

- **窗口平滑弹出**：通过 GSAP 的 `scale`、`opacity` 和自定义缓动实现
- **拖拽效果**：使用 GSAP Draggable 插件实现可拖拽窗口
- **视差滚动**：使用 GSAP ScrollTrigger 实现视差与滚动联动

### Rive：设计师驱动的交互动画

[Rive](https://rive.app/) 是 Dropbox Brand Guidelines（Awwwards SOTD）等作品使用的交互动画工具。设计师在可视化编辑器中制作动画，导出 `.riv` 文件后在 React 中通过 `@rive-app/react-webgl2` 播放。

- **适用场景**：导航菜单、图标动画、状态机交互、品牌展示模块
- **与 GSAP 分工**：Rive 负责设计感强的交互模块；GSAP 负责窗口级、滚动驱动的动画

## 💡 开发哲学

- **窗口独立性**：每个窗口高内聚，不依赖其他窗口的内部状态
- **配置共享**：UI 风格通过 Tailwind 配置统一管理
- **动画分层**：Rive 负责设计师主导的交互模块；GSAP 负责窗口级、滚动驱动的动画；Magic UI / Motion 负责组件级 UI 动画
- **AI 辅助**：使用 Cursor 进行逻辑生成与 UI 优化

## 第一个窗口：登录界面

### 设计灵感

模仿 **Nav Menu from Dropbox Brand** 的前端界面，其初始状态为：纯白背景上几条蓝色细线交织成网格，中央方形区域作为内容区。滚动鼠标后，蓝色细线向中心收缩，中间方形区域缩小，同时发生**颜色镜像变换**（蓝字变白、白底变蓝），形成强烈的视觉聚焦与品牌记忆点。

### 设计叙事：从品牌到登录

采用**两段式叙事**，用动画串联品牌认知与功能入口：

| 阶段 | 视觉状态 | 用户心智 |
|------|----------|----------|
| **第一幕** | 白底 + 蓝线网格 + 中央方形（品牌名 + 简介） | 品牌认知、建立信任 |
| **第二幕** | 网格收缩、区域缩小、颜色反转 | 视觉聚焦、进入「登录空间」 |
| **第三幕** | 蓝色框架内呈现登录表单 | 从品牌过渡到功能，完成转化 |

### 登录模块功能清单

在「高级感」前提下，登录模块应包含以下元素：

**核心功能**

| 元素 | 说明 | 实现建议 |
|------|------|----------|
| 账号输入 | 邮箱或用户名 | shadcn Input，带占位符和校验 |
| 密码输入 | 密码输入框 | 带显示/隐藏切换 |
| 登录按钮 | 主操作 | 品牌色，可配合 Magic UI 的 Shimmer/Ripple |
| 记住我 | 可选 | Checkbox，持久化登录状态 |
| 忘记密码 | 链接 | 跳转或弹窗 |

**增强体验**

| 元素 | 说明 | 实现建议 |
|------|------|----------|
| 第三方登录 | Google / GitHub 等 | 图标按钮，主按钮下方 |
| 注册入口 | 「还没有账号？立即注册」 | 文字链接 |
| 加载状态 | 提交中 | 按钮 loading、禁用 |
| 错误提示 | 账号/密码错误 | Toast 或行内提示 |

**品牌与氛围**

| 元素 | 说明 |
|------|------|
| Logo / 名称 | 顶部，与第一幕品牌名呼应 |
| 副标题 | 如「欢迎回来」 |
| 分隔线 | 区分主登录与第三方登录 |

### 动画与交互节奏

1. **第一幕**：品牌名 + 简介，网格铺满，白底蓝字
2. **滚动触发**：GSAP ScrollTrigger 驱动网格收缩、中心区域缩小、颜色反转（蓝底白字）
3. **第二幕**：中心区域从「品牌文案」过渡到「登录表单」，表单元素可做 stagger 依次出现

### 技术分工

- **GSAP**：网格收缩、区域缩放、颜色反转、滚动联动
- **shadcn/ui**：表单组件（Input、Button、Checkbox）
- **Magic UI**：按钮微动效（Shimmer、Ripple）
