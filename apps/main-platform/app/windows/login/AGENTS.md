# AGENTS.md（Window 1：登录）

## 作用域
本文件作用于 `apps/main-platform/app/windows/login` 目录树。

## 模块职责
- `LoginIntroWindow.tsx`：登录前引导与阶段动画。
- `LoginForm.tsx`：登录/注册表单交互。
- `utils.ts`：登录窗口工具函数。

## 开发规则
1. 阶段切换逻辑（intro -> form）需集中管理，避免状态分散。
2. 注册/登录分支复用同一表单骨架，避免重复实现。
3. 动画层不可影响表单可访问性与交互可用性。

## 扩展性审查重点
- 新增认证方式（如验证码/SSO）是否可插拔；
- 文案与状态是否可配置，而非写死在多处；
- 动画与业务逻辑是否保持解耦。
